# Zombie Waves

**Zombie Waves** is a first-person survival shooter (FPS) playable in both classic desktop mode and Virtual Reality (VR). The player is trapped inside a dark arena, forced to face endless waves of zombies that progressively grow in number, speed, and difficulty with each passing wave. It is perfect for fans of fast-paced shooters and anyone looking for an immersive action experience.

## 🛠️ Requirements & Installation

To run or modify this project, you need to have **Unity** installed on your system. 

### 1. Install Unity via Unity Hub
1. Download and install **Unity Hub** from the official website.
2. Open Unity Hub, go to the **Installs** tab, and install the specific Unity Editor version used for this project (e.g., *Unity 2022.3 LTS* or newer). Make sure to include the **Windows Build Support** module during installation.

### 2. Import the Project Package
Once Unity is successfully installed on your computer, you can import the project simply double-click the provided `.unitypackage` file. Unity Hub will automatically launch and prompt you to import the project directly into your workspace.
